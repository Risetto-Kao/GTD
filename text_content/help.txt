a. GTD是什麼
   歡迎來到 GTD (Getting Things Done，中譯：搞定)
   這是 David Allen 先生創作出來為人生管理、專案管理及時間管理的一套方法及工具
   我們大略上簡化了他的一些概念，希望透過這個小程式來協助您管理好自己的計畫
   無論是工作計畫、學習計畫、出去玩計畫，又或者是簡單的待辦事項清單、家事清單等等
   由大到小的協助您管理好時間，追逐自己的理想生活
   同時我們也設計了一些自訂的獎勵機制，讓養成習慣的門檻降低，也可以更有效率的執行跟規劃
  
b. 獎勵系統
   為了降低使用門檻，有更舒適的體驗跟養成習慣的動機，我們設計了一套獎勵回饋機制
   我們在建立任務時都會給自己的任務設定一個點數，完成後點數便會累積
   作為犒賞自己完成任務、管理好時間跟專案的回饋

c. 輸入指令
   '-i' => initTarget: 可以新增一個大方向的目標，比如：出國留學、期末報告、家事、出國旅行。
   '-s' => show events: 可以展現出目前的事項安排，會由我們設計好的版面呈現，一目了然！
   '-a' => add: 可以新增新的事項在目標清單裡面。
   '-d' => done: 完成事項之後，可以藉由 done 把完成的事項打勾，方便檢視目前的進度狀況
                 完成事項後點數就會自動累積到目前的點數上面了。
   '-ar'=> add reward: 可以藉由此功能增加自己的獎賞清單，設定點數以便自己可以兌換獎賞。
   '-sr'=> show reward list: 可以把自己目前的獎賞清單列舉出來，查看目前有什麼獎賞在清單中。
   '-gr'=> gain reward: 可以根據現在自己的積分兌換獎賞犒賞自己。
   '-c' => check time: 確認有哪些已經逾期的任務
   '-save' => save file: 存檔
   '-exit' => exit this program: 結束程式
